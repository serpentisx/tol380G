Use arrow keys to move the paddle and spacebar to fire the ball


- Powerups drops when a brick is destroyed

- There are 5 powerups. The powerups are generated randomly. A brick has 1/4 chance of having a power up

- Superman effect : Goes through bricks and smash every brick in its way, without bouncing back. This effect lasts for 6 seconds

- Increase paddle size : increases the paddle size by 20px

- Decrease paddle size : decreases the paddle size by 20px

- Incrase ball speed : increases the ball speed by 20%

- Decrease ball speed: decreases the ball speed by 20%


- There is a scoring system. For each brick destroyed 1 point is gained

- There is a game over detection. The game will be over when a player destroys all the bricks. The game will also end if a player loses all his life

- There is a trailing effect on the ball
